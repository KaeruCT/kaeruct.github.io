# kaeruct.github.io

Blog built on [Hugo](https://gohugo.io/).

## Run hugo

```
$ hugo server
```

## Create new post

```
$ ./new.sh post-title
```

## Deploy

Will happen automatically when pushing to main, thanks to [this setup](https://gohugo.io/hosting-and-deployment/hosting-on-github/).
